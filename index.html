NTP Server Monitoring - DevOps

Overview

Developed a Python-based monitoring system for NTP servers at NPL, ensuring precise time synchronization for critical applications. The script monitors servers like `samay1.nic.in` and `157.20.66.8`, logging responses and generating CSV reports.

Technologies Used

  • Python
  • Docker
  • Threading
  • Logging
  • CSV

Features

Rate-limited alerts after 3 consecutive failures or high offsets (>0.01s), log rotation, configuration file support, and Docker containerization for deployment.

Flask Alert System - Web Application

Overview

A Flask-based API developed at NPL to send email alerts to multiple recipients using Gmail’s SMTP. Initially designed for SMS, it pivoted to email due to API constraints, tested with cURL in VS Code.

Technologies Used

  • Python
  • Flask
  • SMTP
  • JSON
  • VS Code

Features

Dynamic email recipient lists, secure SMTP configuration with App Password, and robust error handling for network timeouts.
